# ✍️ Script Maker for Creative Dummies

Un editor de guiones simple, rápido y enfocado en la escritura.

Diseñado para eliminar fricción y ayudarte a concentrarte en lo importante: escribir.

---

## 🚀 Qué es

Script Maker es una aplicación web que permite escribir guiones en formato profesional (cine) sin distracciones, con una interfaz limpia y estructurada.

---

## ✨ Features

- 🎬 Formato de guion estándar (escenas, diálogos, transiciones)
- 🧠 Modo Focus para escritura sin distracciones
- 💾 Guardado automático en el navegador (IndexedDB)
- 📚 Biblioteca de guiones local
- 📤 Exportación:
  - JSON (proyectos)
  - TXT (lectura)
  - PDF (formato de guion)
- 🏷 Carátula editable (título, autor, versión, contacto)
- ⚡ Edición rápida con modales
- 🧩 Arquitectura modular (lista para escalar)

---

## 🧱 Stack técnico

- HTML / CSS / JavaScript puro (sin frameworks)
- IndexedDB para almacenamiento local
- Render dinámico del documento
- Exportación vía Blob
